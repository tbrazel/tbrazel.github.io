Using Python in a GitHub repo

  • Navigate to the repo you're working in and set up a virtual environment python3 -m venv venv and then activate it with source venv/bin/activate
  • Install all relevant packages with pip locally in this repository
  • Git ignore venv/
  • Include a list of required packages using pip freeze > requirements.txt. Anyone setting up this code for themselves can create their own venv and use pip install -r requirements.txt
